# AI-Powered Quiz Application 🧠

A fast, interactive quiz application that dynamically generates conceptual questions using the DeepSeek API and provides smart, semantic grading and feedback.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **AI Core:** DeepSeek API

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd Backend
2. Create a .env file in the backend/ folder and add your DeepSeek API key:
    DEEPSEEK_API_KEY=your_actual_api_key_here
3. Start the Flask server:
    python app.py
